1.сделать папку для проекта
2.ssh-keygen -t rsa -b 4096 -C "your_mail@example.com"
3.делаем в папке .ssh файл config
4.вписываем в config -

# Личная учетка "свой ник на гите" на GitHub
Host github.com
   HostName github.com
   User git
   AddKeysToAgent yes
   IdentityFile (путь к файлу ключа rsa)C:\Users\Igor\.ssh\id_rsa

5.делаем форк проекта на сайте на свой аккаунт гитхаб по данной ссылке
https://github.com/ppalmarr/Museum

6.заходим в гитбаш в папке с проектом и выполняем команду
git clone сюда копируем ssh ссылку с сайта своего форка

7.добаваляем удаленный репозиторий, который является главным командой 
git remote add имя любой без пробелов дальше через пробел ssh ссылку на этот репозиторий git@github.com:ppalmarr/Museum.git (мастер репозитрий)

8.выполняем команду git pull имя удаленного репозитория main

9. делаем ветку git checkout -b имя ветки

10. переход между ветками git checkout имя ветки

11. делаем изменения на локальном репозитории

12.фиксируем их в созданной ветке командами git add . и git commit -m "название действия" и git push origin имя ветки, нельзя делать изменения в main

13.когда закончили задачу в данной ветке, заходим на сайт в ветку в которой закончили работу нажимаем на branches далее newpull request выбираем место назначения main palmar пишем коммент + pull request

14. переходим в мейн командной git checkout main

15. делаем git pull имя удаленного репозитория main

16.  выполняем git add . и git commit -m "" и git push origin main

17. создаем ветку git checkout -b имя ветки для задачи

18. работаем в ней и далее по циклу с 11 пунка до этого
